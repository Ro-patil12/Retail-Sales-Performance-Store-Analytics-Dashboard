# Retail-Sales-Performance-Store-Analytics-Dashboard
This project presents a comprehensive analysis of retail sales data using Power BI. The dashboard provides insights into revenue, product performance, store efficiency, and customer purchasing behavior to support business decision-making.
Business Problem : 
Retail businesses need a clear understanding of
- Sales performance across products and stores  
- Customer purchasing patterns  
- Peak sales hours and trends  
- Product contribution to overall revenue  
This dashboard helps stakeholders monitor performance and identify growth opportunities.

Key Metrics :
Total Revenue:$698K
Total Transactions:149K+  
Total Quantity Sold:214K+  
Average Transaction Value:$4.69  

Dashboard Features :
Executive Overview
- Monthly sales trend analysis  
- Product category contribution (Coffee, Tea, Bakery, etc.)  
- KPI cards for quick performance tracking  
Product Insights
- Top 10 products by revenue  
- Category-wise revenue contribution  
- Product-level performance (Revenue, Quantity, Transactions)  
Store Performance
- Sales comparison across store locations  
- Best performing store identification  
- Transactions and quantity sold by store  
Time Analysis
- Hourly sales trend (Peak hours identification)  
- Weekday vs Weekend sales comparison  
- Day-wise performance insights  

Tools & Technologies Used :
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Modeling
- Data Cleaning / Power Query Editor
- MYSQL
- Excel / CSV

Key DAX Calculations : 
- Total Revenue  
- Average Transaction Value  
- Time-based analysis (Hourly, Daily, Monthly)  
- KPI Metrics (Total Transaction, Quantity)
- Peak Hours Label
- Running Total
- Performance to Date (MTD, QTD)
- Month over Month change
- Previous Month Sales

Key Insights :
Coffee category contributes the highest revenue (~38%)  
Barista Espresso** is the top-performing product (~$91K revenue)  
Hell's Kitchen** is the best-performing store (~$236K revenue)  
Peak sales occur during morning hours (8 AM – 10 AM)  
Weekdays generate ~71% of total revenue  

How to Use :
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Refresh the dataset  
4. Use filters to explore insights
